# v1.2 — New Dailies & First-Run Setup

### New Features

- **First-time setup wizard** — A popup appears on first install letting you choose which daily categories to track before any reminders fire. All categories are enabled by default and can be changed later in settings.

- **Ogri'la / Sha'tari Skyguard dailies** — New tracked category with the following quests:
  - The Relic's Emanation
  - Bomb Them Again!
  - Wrangle More Aether Rays!
  - Banish More Demons
  - Fires Over Skettis
  - Escape from Skettis

- **World PvP dailies** — The PvP category now also tracks Hellfire Peninsula, Terokkar Forest, and Nagrand world PvP dailies for both factions:
  - *Horde:* Hellfire Fortifications, Spirits of Auchindoun, Enemies Old and New
  - *Alliance:* Hellfire Fortifications, Spirits of Auchindoun, In Defense of Halaa

- **Per-category toggles** — Enable or disable tracking for each daily category (Dungeons, Heroics, PvP, Ogri'la) independently, both in the first-run setup and in the settings panel.

### Improvements

- Settings panel now includes a "Tracked Categories" section with checkboxes for all four categories.
- Quest checks respect category toggles — disabled categories are skipped entirely.
